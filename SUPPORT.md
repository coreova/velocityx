# VelocityX Support

VelocityX is a free, open-source Coreova browser extension. Public support happens through GitHub.

- Website: https://coreova.github.io/
- Source: https://github.com/coreova/velocityx
- Publisher: Coreova
- Creator: Maruf Ahmed Limon
- License: MIT licensed by Coreova

## Start Here

- Read `README.md`.
- Check existing issues and discussions.
- Confirm you are using a Chromium-based browser that meets the minimum version in `manifest.json`.
- Try reloading the tab and the unpacked extension before opening a bug report.

## Questions

Use GitHub Discussions when they are enabled. Good topics include setup questions, shortcut configuration, site-rule examples, UI feedback, and workflow ideas.

## Bugs

Use GitHub Issues for reproducible bugs:

- https://github.com/coreova/velocityx/issues

Include version, browser, operating system, affected site, reproduction steps, expected behavior, actual behavior, and relevant screenshots or logs.

## Feature Requests

Use the feature request template. Explain the user, workflow, problem, and useful outcome before proposing a solution.

## Security

Do not post vulnerabilities publicly. Follow `SECURITY.md`.

## Support Limits

Public repositories do not imply commercial support, private implementation help, custom development, or guaranteed response times unless a future Coreova support channel explicitly says otherwise.
