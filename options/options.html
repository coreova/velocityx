Help & FAQ

Common Questions

How do regex site rules work?

Write them like /^(.+\\.)?coursera\\.org$/i. VelocityX checks the current hostname first for exact domain rules, then tests regex rules in the order shown on this page.

What is the difference between Custom Overlay CSS and Site Rule CSS?

Custom Overlay CSS styles VelocityX inside its Shadow DOM. Site Rule CSS styles the page itself, which helps when a site's own player needs spacing or z-index fixes.

Can I create modifier shortcuts?

Yes. Open Shortcuts > Custom Shortcut Combos, pick an action, then capture any combination with Ctrl, Alt, Shift, or Meta.
