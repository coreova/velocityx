# Changelog

All notable VelocityX changes should be documented here.

This project follows practical release notes: user-facing changes, bug fixes, privacy/security changes, permission changes, and migration notes should be easy to scan.

## [1.0.0] - 2026-05-03

### Changed

- Updated the extension title to `VelocityX — Video Speed Controller` and refreshed the store-ready summary across source metadata, docs, launch copy, and release assets.
- Prepared VelocityX for Coreova open-source publishing.
- Updated publisher website links to `https://coreova.github.io/` and source links to `https://github.com/coreova/velocityx`.
- Refined the popup, options, DevTools, analytics, and overlay styling with a calmer premium graphite palette.
- Added local analytics share-card generation with PNG save, caption copy, and native share fallback support.
- Added project-specific community, support, security, and governance files.
- Added browser package build guidance and repeatable release ZIP generation.
- Updated the MIT license holder to Coreova and standardized public attribution across `LICENSE`, `AUTHORS.md`, README, support, privacy, governance, and extension UI.
- Added compact website and GitHub repository links in the popup footer and settings sidebar.

### Notes

- No extension permissions changed.
- No playback control logic changed.
- No telemetry, remote analytics, donation prompts, or promotional notifications were added.
